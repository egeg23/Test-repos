## 💡 Примеры промптов для AI

### Пример 1: Создание экрана
```
Создай Flutter экран для списка товаров:

Требования:
- Используй Riverpod для управления состоянием
- Данные загружаются с API (Dio)
- Показывать skeleton loader при загрузке
- Pull-to-refresh
- Карточка товара: фото, название, цена, рейтинг
- При нажатии - переход на детали

Модель товара:
class Product {
  final String id;
  final String name;
  final double price;
  final double? rating;
  final String? imageUrl;
}
```

### Пример 2: Генерация API клиента
```
Создай API клиент на Dio для:

BASE_URL: https://api.seller-ai.ru/v1

Endpoints:
- GET /products - список товаров
- GET /products/{id} - детали товара
- PUT /products/{id}/price - изменить цену

Добавь:
- Interceptor для JWT токена
- Обработку ошибок
- Retry logic
```

### 1. Давай контекст
```
❌ Плохо: "Сделай кнопку"
✅ Хорошо: "Сделать кнопку 'Добавить' в AppBar, 
          цвет #007AFF, при нажатии открыть BottomSheet"
```

### 2. Показывай примеры данных
```
❌ Плохо: "Список товаров"
✅ Хорошо: "Список товаров. Пример:
          {
            id: '123',
            name: 'iPhone 15',
            price: 99900,
            rating: 4.8
          }"
```

### 3. Проси по шагам
```

Шаг 4: Соедини всё
```

```
Ты: "Сделай список"
AI: [генерирует]
Ты: "Добавь поиск сверху"
AI: [добавляет]
Ты: "Сделай поиск real-time"
```
