# Анализ и оптимизация Seller AI Bot

## О проекте
**Seller AI Bot** — автономная система управления продажами на российских маркетплейсах (Wildberries, Ozon, Авито).

### Что делает бот:
1. Пользователь подключает API магазина
2. AI каждые 10 минут проверяет: цены конкурентов, остатки, рекламу
3. AI принимает решения: изменить цену, ставку в рекламе, заказать товар
4. Решения выполняются через API автоматически
5. Система учится на данных
