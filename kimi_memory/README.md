# 🧠 Kimi Memory Backup

## Описание
Эта папка содержит резервную копию критичной памяти и правил работы.

**Создано:** 2026-03-19
**Назначение:** Восстановление контекста после перезагрузки

---

## Как использовать

### При перезагрузке Kimi:
1. Прочитать `WORKFLOW_RULES.md` — основные правила работы
2. Прочитать `MEMORY_SUMMARY.md` — ключевые факты о проекте
3. Проверить `ACTIVE_TASKS.md` — текущие задачи

### Для пользователя:
Если Kimi "забыл" что-то — скажи:
> "Прочитай память из папки memory/"
